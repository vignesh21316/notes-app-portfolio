# NoteCraft

NoteCraft is a simple and polished notes app built with HTML, CSS, and JavaScript.

![NoteCraft preview](assets/notecraft-preview.svg)

## Links

- Repository: [vignesh21316/notes-app-portfolio](https://github.com/vignesh21316/notes-app-portfolio)
- Live demo: [Open NoteCraft](https://vignesh21316.github.io/notes-app-portfolio/)

## Features

- Create, edit, pin, copy, search, sort, and delete notes
- Store notes locally in the browser with LocalStorage
- Use a responsive layout that works across desktop and mobile
- Save notes quickly with the `Ctrl + Enter` shortcut

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Browser LocalStorage

## Run Locally

```bash
git clone https://github.com/vignesh21316/notes-app-portfolio.git
```

Then open `index.html` in your browser.
